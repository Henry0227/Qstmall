首页

QST-mall 项目是一套电商系统，包括 QST-mall 商城系统及 QST-mall-admin 商城后台管理系统， 基于 Spring Boot 2.1及相关技术栈开发。

    前台商城系统
  • 首页门户
  • 商品分类
  • 新品上线
  • 首页轮播
  • 商品推荐
  • 商品搜索
  • 商品展示
  • 购物车
  • 订单结算
  • 订单流程
  • 个人订单管理
  • 会员中心
    后台管理系统
  • 轮播图管理
  • 热销商品管理
  • 新品上线管理
  • 为你推荐管理
  • 商品管理
  • 订单管理
  • 会员管理
  • 分类管理
  • 修改密码
